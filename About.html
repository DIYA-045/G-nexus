Project Overview:

This project is a Student Information Management System using HTML and Database

Objective:

To store student details and Display them using Hyperlinks

How it works:

  1. Student names are shown on the home page.

  2. Clicking on the persons Hyperlink passes the student ID.

  3. The System recieves data from the Database

  4. Full details are displayed on Details page

Technologies used:

HTMLCSS,PHP,MYSQL
